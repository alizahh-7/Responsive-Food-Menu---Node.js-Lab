## 📌 About the Project

A fully responsive **Food Menu Webpage** built using **HTML5**, **CSS3**, **Flexbox**, and **CSS Grid**.  
The layout adapts smoothly across desktop, tablet, and mobile devices using media queries.

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| HTML5 | Page structure |
| CSS3 | Styling & animations |
| Flexbox | Alignment & spacing |
| CSS Grid | Multi-column layout |
| Media Queries | Responsive design |
| VS Code | Code editor |
| Google Chrome | Testing & DevTools |

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/alizahh-7/food-menu.git
   ```
2. Open the folder in **VS Code**
3. Double-click `index.html` **or** use the **Live Server** extension
4. Open in **Google Chrome** and test responsiveness with `F12 → DevTools`
